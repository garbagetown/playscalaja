Play Scala enables you to use the Scala language for your application keeping key properties of the Play framework. It uses a more functional and Scala idiomatic style of programming without giving up on simplicity and developer friendliness.
version 0.9 published on April 13 2011  —  play install scala-0.9  —  Check the roadmap to 1.0

Add Scala power to Web applications

A Scala Web framework made by Web developers. Gentle learning curve, start easy, gain in power as you learn more and more about the Scala language.

$ play install scala $ play new myScalaWebapp --with scala $ play run

Scala flavored Play API

Use the "full stack" Play API, enabled for the expressivity and conciseness of Scala language.

Cache.getAsync("headlines", "1h", "5d") { WS.url(endpoint).get().focusOnOk().map( _.getXml() \\ "title" ) }

Expression oriented

A controller method is a function that returns a value. This automatically means composability and easy code reuse.

def show(id: Long) = { Order.findById(id).map( o => Template( 'order -> o, 'user -> connectedUser ) ).getOrElse(NotFound("No such order")) }

Powerful SQL databases access

Anorm is simplification of JDBC with a minimal interface reusing pre-existing Scala interfaces (collections, pattern-matching, parsers combinators).

val postsWithAuthor:List[(Post~User)] = SQL( """ select * from Post p join User u on p.author_id = u.id order by p.postedAt desc """ ).as( Post ~< User * )
